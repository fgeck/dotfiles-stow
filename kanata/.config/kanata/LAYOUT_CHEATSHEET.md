## Text Selection & Movement

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  FORMULA:  Hold Mods (left hand) + Hold Space + Tap Arrow (right hand)     │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Navigation (Hold Space, tap right hand)

| Want              | Hold        | Tap |
|-------------------|-------------|-----|
| → right           | `Space`     | `L` |
| ← left            | `Space`     | `J` |
| ↑ up              | `Space`     | `I` |
| ↓ down            | `Space`     | `K` |
| → word            | `S Space`   | `L` |
| ← word            | `S Space`   | `J` |
| → line end        | `Space`     | `O` |
| ← line start      | `Space`     | `U` |

### Selection (add D for Shift)

| Want                   | Hold           | Tap |
|------------------------|----------------|-----|
| Select → char          | `D Space`      | `L` |
| Select ← char          | `D Space`      | `J` |
| Select → word          | `D S Space`    | `L` |
| Select ← word          | `D S Space`    | `J` |
| Select → to line end   | `D A Space`    | `O` |
| Select ← to line start | `D A Space`    | `U` |
| Select line ↓          | `D Space`      | `K` |
| Select page ↓          | `D Space`      | `P` |

### IDE Line Movement (Shift+Alt = D+S)

| Want            | Hold        | Tap |
|-----------------|-------------|-----|
| Move line ↑     | `D S Space` | `I` |
| Move line ↓     | `D S Space` | `K` |

## Tips & Tricks

### Timing
- **Tap** = Quick press < 200ms → outputs letter
- **Hold** = Press > 200ms → activates modifier/layer
- Too slow? You get modifier. Too fast? You get letter.

### The Golden Rule
```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│    LEFT HAND holds modifiers  ←→  RIGHT HAND taps keys (or vice versa)    │
│                                                                             │
│    NEVER hold and tap with the same hand!                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Troubleshooting

| Problem                | Cause              | Fix                       |
|------------------------|--------------------|---------------------------|
| Get `a` not Cmd        | Tapped too fast    | Hold longer before tap    |
| Get Cmd not `a`        | Held too long      | Tap faster                |
| Wrong layer active     | Forgot to release  | Release all, start over   |
| Same-hand conflict     | e.g., Cmd+S        | Use opposite `;` for Cmd  |
